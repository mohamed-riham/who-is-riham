Comparative Big-Data Fraud Transactions Detection Case Study

Title: A Comparative Study of Random Forest and XGBoost for Detecting Credit Card Fraud Transactions using Big Data

Principal Academic Details: Student ID: 1028401 | Assessor: A.L.F. Sajeetha | Supervisor: Mr. Mohamed Nismy

This rigorous investigation compiles classification statistics across a database of over 1,000,000 imbalanced financial records (containing only 0.52% positive Fraud events against 99.48% legitimate transactions). Key Metrics Accomplished:

  • XGBoost Model Classifier: Precision: 99.71%, Recall: 99.74%, F1-Score: 99.72%, Accuracy: 99.74%
